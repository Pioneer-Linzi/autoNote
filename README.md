# 
这个是用python 脚本写的...
用于自动的生成今日日志模版
进行代码重构，减少冗余代码！
## 功能如下：
    - 自动按年份，月份生成文件夹，
    - 自动从模版中获得模版，并从昨天的日志中获得今日的计划任务
    - 如果前一天没有任务或者没有文档，则循环到直到有文档的那一天，把没有完成的任务自动的加入当天
    - 自动的在文章的头部加入时间
    - 自动的把昨日未完成任务加入到今日任务中
    - 在temp.md中加入summary 标记，为以后提取总结做准备
    - 运行shell 脚本可以自动的生成日志，与自动的与自己的github结合，
    - 把shell 脚本加入开机自启中，可以使开机后自己的生成今日任务，并自动的打开网页
